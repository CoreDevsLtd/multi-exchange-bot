IB Gateway / Client Portal (session login)

Interactive Brokers does not use exchange-style API keys here. The bot’s API calls go to the URL in Base URL above (IB Gateway or TWS with Client Portal Web API).

  1. Set Base URL to that Gateway. On the same PC as the browser: https://localhost:5000 is fine. If the dashboard is hosted on a server, localhost in your browser only reaches your PC — expose Gateway with an HTTPS tunnel (ngrok, Cloudflare Tunnel, etc.), put that URL in Base URL, and use the same URL so the bot can reach it too.
  2. Start IB Gateway or TWS; enable API / Client Portal on the port that matches the URL.
  3. Click Open Gateway login (opens Base URL) — sign in to IBKR — then Test Connection.
